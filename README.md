# California House Price Prediction (From Scratch)

This repository contains a step-by-step implementation of a Deep Neural Network built entirely from scratch using only **NumPy** and **Pandas**. 

## Current Progress
- [x] Data cleaning and exploration
- [x] One-hot encoding for categorical variables
- [x] Custom Train/Test split logic using NumPy
- [x] Custom Batch Normalization function
- [x] Weight initialization (He Initialization)
